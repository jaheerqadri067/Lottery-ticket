# Lottery Ticket App

A simple React-based Lottery Ticket game where random numbers are generated and the user wins if the total sum becomes 15.

##  Features

- Random lottery ticket generation
- Winning condition based on sum
- React Hooks (`useState`)
- Clean UI using Tailwind CSS

##  Tech Stack

- React
- JavaScript
- Tailwind CSS
- Vite
  

## 🎯 Winning Logic

If the sum of all ticket numbers equals `15`, the user wins the lottery.

Example:

```txt
5 + 4 + 6 = 15 ✅
```
